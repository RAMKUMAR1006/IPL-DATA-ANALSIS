 # IPL Data Analysis – Power BI Dashboard
 ## Project Overview
  This project presents an end-to-end Power BI analytics report built using historical Indian Premier League (IPL) data.The dashboard provides deep insights into team performance, player statistics, match outcomes. It is designed for cricket enthusiasts, data analysts, and anyone exploring sports analytics.


# QUESTIONS(KPIs)
## Team Performance

Which team has won the most matches across all seasons

Which teams perform best while batting first vs chasing

How does toss decision (bat/field) influence match outcomes

What are the head-to-head records between major teams

## Player Performance

Who are the top run-scorers in IPL history and season-wise

Which batsmen have the highest strike rate and average

Who are the leading wicket-takers and most economical bowlers

Which all-rounders contribute significantly in both batting & bowling

# Process
Data collection: Gather IPL data - Auction,Match records,Batting stats and Bowling Stats

Data Cleaning & Preparation: Use Excel / Power Query to remove duplicates, merge tables, and create calculated columns

Dashboard Creation: Design interactive dashboards in Power BI and Excel using charts, slicers, and KPI cards to visualize insights.

# Project insights

 Certain teams consistently maintain high win percentages and show dominance in long-term performance.Top batsmen contribute heavily during powerplay and death overs, with aggressive scoring driving match momentum.Key bowlers with strong economy rates and high wicket-taking ability significantly impact match outcomes.Player consistency plays a major role—season trends highlight reliable performers and emerging talents.

# Final Conclusion

  The IPL Data Analysis dashboard delivers clear, data-driven insights into team strategies, player performance, and match outcomes, highlighting the key factors that shape success in the league. It provides a valuable foundation for informed decision-making and deeper analytical exploration of IPL trends.
